# Buyer Readiness Checklist

12–24 month prep before a serious process. **Companies are bought, not sold** — readiness enables inbound leverage.

## Scorecard (Rate 1–5 Each)

| Area | Ready (5) | Gap (1–2) |
|---|---|---|
| **Financials** | 36 mo GAAP-ish P&L; cohort retention export | Spreadsheets only; no cohort view |
| **Metrics** | NRR, GRR, CAC, LTV, burn multiple monthly | ARR only |
| **Legal** | All IP assigned; clean cap table | Missing contractor IP |
| **Commercial** | No customer >15% ARR; documented playbook | Founder-only sales |
| **Technical** | SOC2 or equivalent; DR documented | Security questionnaire fails |
| **Team** | VP layer; succession plan | Key-person risk |
| **Narrative** | Clear category + why now for buyer | "We're for sale" signaling |

**Go threshold:** Average ≥4.0 and no area at 1. Use `templates/exit-readiness-scorecard.md` for full worksheet.

## 24-Month Timeline

### Months 1–6: Clean up

- [ ] Cap table audit (`equity-management`)
- [ ] IP assignment sweep — all employees, contractors, agencies
- [ ] Customer concentration plan — no logo >15% by month 12
- [ ] Separate services from subscription revenue in reporting
- [ ] Implement monthly metrics pack (`due-diligence-metrics-pack.md`)

### Months 7–12: Build proof

- [ ] NRR trend ≥110% or documented improvement plan
- [ ] GTM repeatability — AE quota attainment >60% of team
- [ ] Founder dependency: founder closes <30% of new ARR
- [ ] SOC2 Type I or security doc equivalent for enterprise buyers
- [ ] Strategic partnerships (not "shopping" — relationship building)

### Months 13–18: Market signals

- [ ] Accelerating or stable growth 2+ quarters
- [ ] Industry recognition, case studies, analyst mentions
- [ ] Optional: banker conversation for $20M+ ARR
- [ ] Data room 80% complete before first LOI

### Months 19–24: Process (if inbound)

- [ ] Counsel engaged; NDAs template ready
- [ ] Management presentation draft
- [ ] Diligence metrics pack current within 30 days
- [ ] Employee retention plan for close

## Due Diligence Package (Minimum)

| Category | Documents |
|---|---|
| Financial | 36 mo P&L, BS, CF; ARR bridge; cohort retention |
| Metrics | NRR, GRR, churn, CAC, LTV, magic number, burn multiple |
| Legal | Cap table, charter, IP assignments, material contracts |
| Commercial | Top 20 customers, pipeline, churn reasons, pricing |
| Technical | Architecture, SOC2, dependency inventory, DR |
| HR | Org chart, key employment agreements, option pool |

Full spec → `due-diligence-metrics-pack.md`.

## Buyer-Type Prep

| Buyer | Extra prep |
|---|---|
| Strategic | Integration map; customer overlap analysis; product roadmap 18 mo |
| PE | EBITDA bridge; 24-mo margin expansion plan; GTM efficiency metrics |
| Bootstrap acquirer | SDE reconciliation; owner salary normalization; simple data room |
| Acqui-hire | Team bios; tech stack; velocity metrics |

## Do Not

- Announce "for sale" broadly — signals weakness
- Optimize vanity ARR — diligence strips one-time revenue
- Start process with NRR <100% and no fix — multiple compression at LOI
- Skip earnout modeling — count probability-weighted, not 100%
