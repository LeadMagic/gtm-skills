# Founder Comp Budget — Payroll Guardrails

**Last updated:** June 2025 · **Review:** H1 each year against Bridge Group + runway model

## Sales Payroll as % of ARR

| ARR | Target sales payroll % | Hard ceiling | Notes |
|---|---:|---:|---|
| $0–$500K | 0% FTE sales | 10% | Founder sells; contractors only |
| $500K–$1M | 10–20% | 25% | 1 AE max |
| $1M–$3M | 15–30% | 35% | AE + SDR or 2 AEs |
| $3M–$8M | 20–35% | 40% | Manager optional |
| $8M–$20M | 25–40% | 45% | VP + pod |
| $20M+ | Model with board | — | CRO economics |

**Sales payroll** = base + taxes + benefits + target variable + employer tool cost per rep.

## Runway Check (Before Any Offer)

```
months_runway_after_hire =
  cash_balance / (monthly_burn + new_hire_fully_loaded_monthly_cost - expected_monthly_new_ARR × gross_margin)
```

| Runway after hire | Decision |
|---|---|
| <9 months | Delay hire or contractor |
| 9–15 months | Hire only if pipeline proves payback <6 mo |
| 15+ months | Proceed with written comp plan |

Load `financial-modeling` for full scenario.

## Variable Accrual Reserve

Accrue **100% of target variable** for all quota carriers in financial plan — even if attainment is 80%. Under-accrual kills runway in Q4.

## Equity Budget (Seed–Series A)

| Total option pool remaining | Typical first sales grants |
|---|---|
| 10–15% FD unallocated | First AE 0.25–0.75%; VP 0.5–1.25% |

Board approval required above policy band.

## Founder Compensation (Don't Ignore)

Pay yourself enough to not negotiate from desperation:

| Stage | Founder salary guidance |
|---|---|
| Pre-revenue | Minimal living + equity focus |
| $500K–$2M ARR | $80–120K if bootstrapped |
| Raised seed/A | Market-ish $120–180K |
| $5M+ ARR | Board-set CEO comp |

Underpaid founders overpay hires or accept bad terms — fix founder comp first.

## Negotiation Budget Line

Set explicit **negotiation reserve** before finals:

| Role | Typical reserve |
|---|---|
| AE | +$5–15K OTE equivalent (equity/signing/ramp) |
| SDR | +$3–8K |
| VP | +$20–40K total cash or equity refresh |

If reserve exhausted, walk — don't break 5:1 or runway.
