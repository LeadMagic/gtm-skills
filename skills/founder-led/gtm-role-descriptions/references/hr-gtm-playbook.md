**Frameworks for GTM teams**

| Framework | GTM application |
|---|---|
| **Culture-first onboarding** | Week 1 = access + norms + shadow path — not product trivia alone |
| **Onboarding survey → action** | Day 14 / 30 / 60 pulse; manager closes loop in <1 week |
| **New manager training** | Promote SDR→manager only after manager curriculum (coaching ≠ top rep) |
| **Role-specific onboarding** | SDR, AE, RevOps get different certification gates — same HR process |
| **Remote/time-zone empathy** | Async docs, recorded shadows, explicit overlap hours for global pods |

**Contrast:** Nordwall = **employee experience systems**. Roberge = **ramp metrics machine**. Bridge Group = **comp math**. Use all three — don't conflate onboarding vibes with quota attainment.

## Revenue hiring system (HR + GTM)

### 1. Pre-hire

| Step | Owner | Artifact |
|---|---|---|
| Role charter + quota math | RevOps + hiring manager | `gtm-role-descriptions` JD |
| Comp band approval | Founder/CFO | `comp-benchmarks.md` · `founder-comp-playbook` |
| Structured interview pack | Hiring manager | `hiring-by-role` scorecards |
| Classification check (SDR/AE) | HR + counsel | `employment-compliance` — **W-2 default** |
| Background / references | HR | Betts/Roberge attainment verification |

### 2. Offer & close

- Document OTE, quota, draw (if any), clawback, equity, geo policy
- **Quota:OTE ~5:1** — reconcile all comp skills via `references/benchmark-reconciliation.md`
- Promotion path (SDR→AE) in offer letter appendix

### 3. Pre-start (HRIS)

| Day −14 to −1 | System |
|---|---|
| Offer signed | HRIS (Rippling, Gusto, Deel for intl) |
| Equipment + MFA | IT + `revenue-team-onboarding` security checklist |
| CRM/sequencer provisioned | RevOps — **day 0**, not week 2 |
| Manager 30-60-90 drafted | Hiring manager + Nordwall onboarding outcomes |

**HRIS for sales orgs:** Rippling/Deel/Gusto for payroll; Culture Amp/Lattice for engagement; Ashby/Greenhouse for ATS — pick one ATS, one HRIS; don't duplicate comp data.

### 4. Ramp (HR measures system, GTM measures output)

| HR metric | GTM metric | Source |
|---|---|---|
| Onboarding survey scores | Days to first qualified meeting | Nordwall + Roberge |
| Manager training completion | Certification pass date | Culture Amp + `revenue-team-onboarding` |
| 30/60/60 check-in completion | 90-day attainment % | `ramp-benchmarks.md` |
| regrettable attrition @ 6 mo | SQO/booking vs tenured | Bridge Group |

**Canonical ramp timelines:** `revenue-team-onboarding/references/ramp-benchmarks.md` — not conflicting 30-60-90 *plan* vs 4–6 month *full productivity* (see benchmark-reconciliation).

## Remote & distributed GTM teams

| Practice | Why |
|---|---|
| Document "golden hours" per region | Pipeline handoffs don't stall |
| Record top rep calls (Gong) for async shadow | Nordwall: connect across time zones |
| Slack channel charter | `skills/management-leadership/revenue-team-onboarding/references/slack-engagement.md` |
| Quarterly in-person for enterprise AEs | Relationship selling still matters |
| Single comp geo band or explicit tiers | Avoid opaque "we'll figure it out" |
