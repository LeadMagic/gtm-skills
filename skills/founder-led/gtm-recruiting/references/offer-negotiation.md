## Founder-Specific Rules

Founders negotiate from **runway + earnability**, not unlimited cash.

| Constraint | Tactic |
|---|---|
| Cash tight | Equity, signing bonus, ramp, territory — not fake high OTE |
| First hire | Publish narrow band; +10–20% reserve for unicorn only |
| Can't match BigCo | Show accelerators + growth equity story with FD% math |
| Candidate deadline | Pre-draft counter before finals; 24h written turnaround |

**Load:** `founder-comp-playbook/references/founder-negotiation-scripts.md`

**Ask candidates:** attainment last 4Q, realized cash today, deadline on competing offer.

## Step 1 — Full Package on the Table

Before negotiating any line item, document:

| Field | Employer provides | Candidate requests |
|---|---|---|
| Base salary | ✓ | |
| Variable / OTE | ✓ | |
| Annual quota | ✓ | Historical team attainment % |
| Ramp / draw | ✓ | |
| Accelerators + cap | ✓ | |
| Clawback policy | ✓ | |
| Equity (% FD, vest) | ✓ | |
| Signing bonus | Optional | If leaving forfeited commission |
| Territory / accounts | ✓ | |
| Start date | ✓ | |

**Rule:** Get comp plan in writing. Verbal promises don't pay commission.

## Step 2 — Anchor on Total Package

Example AE:

| Package A | Package B |
|---|---|
| $70K base / $140K OTE | $80K base / $140K OTE |
| $700K quota (5:1) | $1.4M quota (10:1) |

Package A is earnable; Package B looks higher OTE but same variable with 2× quota.

**Employer:** Always show quota:OTE ratio when discussing OTE changes.  
**Candidate:** Model earnings at 80%, 100%, 120% attainment before accepting.

## Step 4 — Betts-Style Employer Rules

1. **Unicorn / first AE:** Budget +10–20% above target comp band
2. **Poaching employed rep:** Match or beat current **cash**; win on equity, role, trajectory
3. **Geo transparency:** Remote band stated in offer — no post-offer downgrade
4. **Speed:** Written offer within 48–72h of verbal yes

## Step 5 — Candidate Negotiation Script

> "Thank you for the offer — I'm excited about [specific reason]. Based on
> [Bridge Group / RepVue / comparable offers] for [role] in [market], the
> typical OTE is $X–$Y. I'd like to discuss [specific element] to align with
> the market and my last 4Q attainment of [Z%]. I'm also keen to confirm
> quota attainability — could you share team attainment distribution?"

**Tone:** Data + curiosity, not ultimatum. Negotiation skill is part of the sales interview.

## Step 6 — Counter-Offer Matrix (Employer)

| Ask | If budget tight | If budget OK |
|---|---|---|
| +$10K base | +$5K base + $5K signing | Approve base |
| +$15K OTE | Hold OTE; lower quota proportionally | Raise OTE + quota (5:1) |
| More equity | Explain FD%; offer option refresh at 12mo | Increase grant |
| Higher quota relief | Steeper ramp Q1–Q2 | Approve ramp |

## Step 7 — Close Checklist

- [ ] Written offer + comp plan attached
- [ ] Candidate acknowledged quota and ramp
- [ ] Equity grant numbers in offer letter
- [ ] employment-compliance review complete
- [ ] Start date + background check timeline
- [ ] Competing offer deadline noted — respond before it expires

## Red Flags (Either Side)

| Signal | Action |
|---|---|
| Candidate won't share attainment | Pass |
| Employer won't share team attainment | Candidate caution |
| Quota 2× peer OTE without premium | Redesign plan |
| "We'll fix comp after you prove it" | Get it in writing or walk |
