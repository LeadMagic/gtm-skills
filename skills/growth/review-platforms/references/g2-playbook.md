# G2 Playbook

## Profile

- Primary category + up to 3 secondary
- 5+ screenshots updated quarterly
- Video demo <3 min
- Pricing link live
- Feature list matches product (no aspirational)

## Grid Factors

- **Satisfaction:** star rating + review text sentiment
- **Market presence:** traffic, social, employee growth (G2 algorithm)
- **Review recency:** last 90 days weighted heavily

## Generation

- Direct G2 review link per customer (from G2 vendor portal)
- CSM personal ask within 48h of milestone
- Target 2–4 new reviews/month

## Badges

- Use only **current** season badges on site
- Leader / High Performer / Momentum — verify category

## G2 Buyer Intent (paid)

- Route to CRM with source `G2_Intent`
- SLA: SDR contact <24h for high-intent accounts

## Responses

- <48h for all reviews
- Sign with name + title
- Take detailed issues offline via support@
