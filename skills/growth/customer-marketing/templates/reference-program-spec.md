# Reference Program Spec

**Owner:** CS + Sales · **CRM admin:** RevOps

---

## Eligibility

| Criterion | Required |
|---|---|
| NPS ≥ 8 or CSAT equivalent | ☐ |
| Tenure ≥ 6 months | ☐ |
| Health score green | ☐ |
| No open P1 support issues | ☐ |
| Executive sponsor identified | ☐ |

## Limits

| Rule | Value |
|---|---|
| Max reference calls / customer / quarter | 3 |
| Max calls / customer / year | 8 |
| Cooldown after call | 30 days |

## Rewards per tier

| Calls completed | Reward |
|---|---|
| 1–2 / year | Priority support queue |
| 3–5 / year | Swag + product roadmap preview |
| 6+ / year | Advisory invite consideration |

Track spend: `gtm-spend-management` — cap annual advocacy budget.

## AE workflow

1. Check CRM `referenceable` = Y and `reference_calls_ytd` < cap
2. CSM approves (async Slack)
3. Schedule 30-min call; send prep doc to reference
4. Log call + outcome on opportunity
5. Thank-you within 24h

## Rotation

Quarterly report: top 10 over-used references → pause new bookings.
