## Coaching Type → Lead Expert

| Situation | Lead with | Skill |
|---|---|---|
| Rep missing quota | REKS (Jacco van der Kooij) | This skill |
| Weak discovery | SPICED + Barrows drills | `meeting-prep` |
| Deal won't advance | MEDDICC evidence (Whyte) | `pipeline-management` |
| "Need to think about it" | JOLT (Dixon/McKenna) | `buyer-indecision` |
| SDR low reply rate | PVP (Jason Bay) | `cold-email-strategy` |
| SDR low connect / weak cold calls | 3×3 + sell-the-meeting (Slocum) + CallBlitz (Reisert) | `cold-calling`, this skill |
| SDR wrong daily priorities | CRM Activity Buckets (Reisert) | `cold-calling`, `pipeline-management` |
| Phone program diagnosis | Disposition Science (Gilkey) | `cold-calling`, `pipeline-management` |
| Manager not coaching | Dorsey cadence | This skill + `gtm-leadership` |
| Founder still selling | Founder coaching guide | `references/founder-coaching.md` |
| Enterprise deal, weak trust | Three Plays + relationship map (Seidl) | `references/randy-seidl-relationship-selling.md` |
| Champion-only risk | Multi-thread + trust scorecard (Seidl) | `multi-thread-orchestration` |
| Status-quo / "we're fine" | Challenger reframe (Adamson) + Eat Their Lunch (Iannarino) | `objection-handling`, this skill |
| Rep habit / accountability gap | Iannarino discipline + Dorsey cadence | This skill |
