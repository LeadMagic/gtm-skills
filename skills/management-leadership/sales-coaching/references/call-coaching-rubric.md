# Call Coaching Rubric

**Cadence:** 2–3 calls per rep per week (manager or senior AE). **15 min review + 10 min role-play.**

## Scoring (1–5 per dimension)

| Dimension | 1 (Poor) | 3 (OK) | 5 (Excellent) |
|---|---|---|---|
| **Prep** | No research | Basic account notes | SPICED pre-call plan |
| **Discovery** | Pitch-first | Some questions | Impact + Critical Event |
| **Talk ratio** | >70% rep | ~50% | <40% rep; buyer talks |
| **Value** | Feature dump | Benefit statements | Quantified Impact |
| **Objections** | Defensive | Acknowledge + respond | Reframe + confirm |
| **Next step** | Vague | Date proposed | Mutual action plan |
| **CRM** | Not updated | Partial | SPICED + MEDDICC fields |

## SDR / Outbound Additions (Jason Bay — PVP)

| Dimension | Coach toward |
|---|---|
| Relevance | Permissionless value in first 2 lines |
| Personalization | Specific trigger cited |
| Ask | One clear CTA — not three options |

## Demo Calls (Jon Barrows)

| Dimension | Coach toward |
|---|---|
| Agenda | Mutual agenda first 2 min |
| Demo | Pain-linked features only — not tour |
| Checkpoints | "Does this match what you need?" every 5 min |
| Close | Explicit next step before leaving call |

## Feedback Format (Keith Rosen)

**One KEEP:** "At 4:32 you asked about Impact — that opened budget talk. Do more."  
**One FIX:** "At 8:10 you jumped to pricing before Decision Process — next call, ask who else approves."

Rep commits: "I will [fix] on next call with [account] by [date]."

## Gong / Recording Hygiene

- Tag calls for review in weekly 1:1
- Manager comments at timestamp — rep replies with plan
- Top call of week shared in `#wins` (with permission)
