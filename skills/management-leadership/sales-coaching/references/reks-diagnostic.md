# REKS Diagnostic — Jacco van der Kooij / Winning by Design

Diagnose **before** coaching. Coaching the wrong layer wastes weeks.

## REKS Layers

```
Results  ← what happened (lagging)
   ↑
Efforts  ← activity, pipeline coverage (leading)
   ↑
Knowledge ← product, ICP, process, playbook
   ↑
Skills   ← conversation execution (discovery, demo, negotiate)
```

## Diagnostic Questions

| Layer | Manager asks | Red flag |
|---|---|---|
| **Results** | Attainment last 4Q? Win rate trend? | <70% 2+ quarters |
| **Efforts** | Meetings held vs plan? Pipeline coverage ×3? | Activity OK but no results → Skills/Knowledge |
| **Knowledge** | Explain ICP tier A vs B? Product differentiation? | Can't articulate without slides |
| **Skills** | Review last 3 calls — discovery quality? | Talk ratio >60%, no Impact question |

## Intervention Matrix

| Diagnosis | Intervention | Owner | Timeline |
|---|---|---|---|
| Efforts gap | Activity plan, calendar audit, accountability | Manager + rep | 2 weeks |
| Knowledge gap | Enablement module, certification quiz | Enablement | 1–2 weeks |
| Skills gap | Call coaching 3×/week, role-play | Manager | 3–4 weeks |
| Results + adequate E/K/S | Territory review, quota review, fit | Manager + RevOps | Immediate |

## Study Top Performers

Before coaching underperformers, document what **top quartile** reps do differently:
- Calls/week, SQO quality, MEDDICC scores at same stage
- SPICED fields populated in CRM
- Use as coaching benchmark — not vague "be more like Sarah"

## Connection to PIP

If 30-day coaching plan (`coaching-plan-30-day.md`) shows no leading indicator improvement → `gtm-leadership` PIP path. REKS proves you coached the right layer first.
