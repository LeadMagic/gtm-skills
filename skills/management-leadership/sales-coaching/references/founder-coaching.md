# Founder-as-Coach

Before VP Sales or player-coach manager, **the founder is the coach**.

## By Team Size

| Team | Founder coaching time | Focus |
|---|---|---|
| Founder only | 100% on own calls | Document SPICED in CRM |
| 1 AE | 5–8 hrs/week | Shadow calls, weekly deal review |
| 2–3 AEs | 3–5 hrs/week | Deal review + 1 call/rep/week |
| Manager hired | 2 hrs/week | Meta-coach manager on REKS |

## Founder Weekly Calendar (minimum)

| Block | Activity |
|---|---|
| 2× 60 min | Deal reviews with each rep (MEDDICC) |
| 1× 30 min | Pipeline inspection — Commit deals only |
| 1× 60 min | Listen to 1 recorded call per rep — one keep / one fix |

## What Founders Coach (in order)

1. **ICP discipline** — who we don't sell to  
2. **SPICED discovery** — Impact and Critical Event  
3. **CRM hygiene** — if fields empty, coaching doesn't scale  
4. **MEDDICC evidence** — before forecast trust  
5. **JOLT** — when founder still closes enterprise deals  

## Founder Traps

| Trap | Fix |
|---|---|
| Closing every deal yourself | Rep shadows; founder listens more each month |
| Only coaching wins | Review lost deals — REKS on losses |
| No written playbook | `sales-enablement` before hiring AE #2 |
| Skipping 1:1s in fundraise crunch | Calendar immovable — team attrition costs more |

## Handoff to Manager

When player-coach manager starts:
- Founder attends manager's deal reviews for 4 weeks (meta-coach)
- Manager owns call coaching; founder owns 1 strategic deal/month
- REKS diagnostic shared — same language as `gtm-leadership`
