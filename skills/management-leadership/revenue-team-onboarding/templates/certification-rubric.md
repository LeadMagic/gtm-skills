## Gate 3: Mock call (required)

Use `sales-coaching` call rubric. **Minimum 4.0/5.0** to pass.

| Dimension | 1 (weak) | 3 (acceptable) | 5 (strong) | Score |
|---|---|---|---|---:|
| **Opening** | Generic pitch | Relevant hook | Tied to research / trigger |  |
| **Discovery** | Talks > listens | Some SPICED | Full SPICED; no premature demo |  |
| **Value** | Feature dump | One benefit | Quantified outcome |  |
| **Objections** | Defensive | Acknowledges | Reframes with proof |  |
| **Next step** | Vague | Date proposed | Mutual plan + calendar |  |
| **Total** |  |  |  | /25 → /5 |

**SDR add-on:** PVP messaging quality (Jason Bay) — peer review ☐

**Gate pass:** ☐ Score ≥4.0
