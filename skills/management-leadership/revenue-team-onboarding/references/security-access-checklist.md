# Security Access Checklist — GTM Hires

## Day 0 (before customer contact)

- [ ] Corporate email + MFA
- [ ] Slack (standard channels only)
- [ ] CRM role: least privilege (no admin)
- [ ] Calendar + video conferencing
- [ ] Security awareness training assigned
- [ ] Device MDM enrolled
- [ ] Password manager / SSO

## Day 1–3

- [ ] Sequencer (if SDR/AE)
- [ ] Enrichment tools (credit limits set)
- [ ] Gong / recorder
- [ ] Knowledge base / Notion
- [ ] VPN if required

## Restricted until certified

- Production admin panels
- Bulk export / API keys
- Customer PII offline download
- Billing systems

## Offboarding (within 24h)

- [ ] Disable SSO all tools
- [ ] Rotate shared inboxes
- [ ] Remove from sequences
- [ ] Archive Slack; export if required

Align with `soc2-compliance` access control policy.

**Rep hygiene:** Password manager, MFA priority list, phishing patterns, demo
screen-share prep → `references/gtm-security-hygiene-basics.md`.

**Customer data:** GTM hires must read `references/gtm-data-exchange-playbook.md`
before requesting exports from customers.
