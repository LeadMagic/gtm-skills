## Comp Model by Stage (what to implement)

| Stage | Implement | Defer |
|---|---|---|
| **$0–$2M** | Single AE plan; optional draw; equity table | SDR comp complexity, accelerators, manager override |
| **$2–$10M** | SDR + AE plans; 5:1; clawback; ramp table | Multi-year kicker, global geo matrix |
| **$10–$50M** | Manager override; CSM NRR; SPIF process; President's Club | Consumption CRO gates (unless usage revenue) |
| **$50M+** | Consumption/usage gates; geo policy; annual comp committee | Ad-hoc SPIFs without finance review |
