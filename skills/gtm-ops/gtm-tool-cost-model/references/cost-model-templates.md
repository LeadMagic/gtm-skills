# GTM Tool Cost Formulas

## Per-Seat Annual

```
annual = platform_fee + (seats × monthly_seat × 12)
```

## Credit-Based

```
annual = platform_fee + (monthly_rows × credits_per_row × credit_price × 12)
```

## Fully Loaded TCO

```
TCO_year1 = licenses + credits + implementation + integration_hours×rate + cloud
TCO_year2 = licenses + credits + renewal_uplift(%) + maintenance
```

## Cost Metrics

```
tool_spend_pct_ARR = total_GTM_tools / ARR
cost_per_AE = total_sales_tools / AE_count
cost_per_qualified_meeting = stack_cost_SDR / SQOs_per_month
```

## Consolidation Savings

Flag duplicate: enrichment (2+ vendors), intent (2+), conversation intel (2+).

Typical save: 15–30% on renewals with true-down + duplicate cut.
