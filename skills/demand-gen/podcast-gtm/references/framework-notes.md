# Podcast GTM — Framework Notes

Reference tables for `SKILL.md`. Apply named frameworks to justify recommendations — do not cite as decoration.

## Primary frameworks

- **Chris Walker — Demand creation & dark social** — Podcast drives unmeasurable influence; track "heard on podcast" in CRM; 90-day program evaluation.
- **Rand Fishkin — SparkToro audience research** — Ask ICP which shows they listen to; Tier 1 = ICP overlap, not raw downloads (playbook §2 influence map).
- **Ahrefs / SEO playbook §6 — Podcast tour** — Show-note backlinks; one appearance → repurposed content cluster.

## Show tiering (ICP over audience size)

| Tier | Criteria | Expected outcome |
|---|---|---|
| **Tier 1** | ICP listens (even 500 downloads) | Highest pipeline conversion |
| **Tier 2** | Industry shows, quality production, DR backlinks | SEO + credibility |
| **Tier 3** | General business / startup reach | Brand awareness |

**Fishkin method:** Customer interviews → Apple/Spotify keyword search → competitor guest audit.

## Booking outreach principles

| Element | Spec |
|---|---|
| Research | Reference 2–3 specific recent episodes |
| Pitch | 3 counterintuitive topics, not generic "love to come on" |
| Credentials | Data, unique experience, contrarian POV |
| Follow-up | Max 2 touches |

## Interview prep (Walker frequency)

| Asset | Purpose |
|---|---|
| 3 key messages | Repeat 2–3× in different forms |
| Signature story | Memorable customer or founder story |
| Counterintuitive take | Shareable clip for dark social |
| Single CTA URL | Free tool > newsletter > book call |

## Repurposing → content stack (Pattern 26)

One 45-min episode → 10+ assets:

| Asset | Channel | SEO / dark social |
|---|---|---|
| Pull-quote graphics | LinkedIn, X | Dark social |
| Thread / post | X, LinkedIn | Frequency |
| Blog expansion | Site | Cluster link to pillar |
| YouTube Short | YouTube | Discovery |
| Show notes backlink | Host site | playbook §6 link |
| "As seen on" | Website | Trust |

## Pipeline conversion

| Tactic | Implementation |
|---|---|
| Podcast-specific landing page | Unique URL in show notes |
| "Heard you on [Podcast]" | SDR/cold email opener |
| Self-reported source | CRM field on discovery calls |
| Co-promotion | You share episode → rebook rate ↑ |

**Benchmark:** Podcast-sourced leads often convert 2–3× average — pre-education effect (Walker).

## Process phases

1. **Target show selection** — SparkToro + customer interviews; tier list
2. **Booking outreach** — Researched pitches, 3 topics each
3. **Interview prep** — Messages, story, CTA, contrarian take
4. **Content repurposing** — 10+ derivative map within 30 days
5. **Pipeline tracking** — UTM, landing page, CRM self-reported field
