## Core thesis: conversations are the content engine

Most B2B teams **manufacture** content (writers, agencies, AI drafts). Lizak's
model **records** content — real conversations with executives, customers, or
peers — then repurposes into the full distribution stack.

| Manufactured content | Conversation-first content |
|---|---|
| Writers guess what resonates | Audience reacts live — instant signal |
| High production cost per asset | Low barrier: hit record on internal expertise |
| Disconnected from relationships | Live chat, comments, and DMs deepen network |
| One-off posts | One live session → podcast + clips + feed posts |

Origin story (public): Lizak started by interviewing her CEO to understand a
complex data business, clipped short videos, then launched **Whiskey WinsDay**
(now **WinsDay**) as a LinkedIn Live show — later scaling Reveting to run 10+
livestream podcasts for B2B clients.

## The Reveting livestream flywheel

```
Plan weekly show → Promote to network → Go live on LinkedIn
        ↑                                        ↓
   Measure relationships              Engage chat + commenters live
        ↑                                        ↓
Repurpose → clips, podcast, posts ← Record + save replay
```

**Phase outputs**

| Phase | Output | Purpose |
|---|---|---|
| Show design | Title, guest, 3-5 talking points | Predictable weekly cadence |
| Pre-live | Event post + DMs to warm contacts | Notification graph + attendance |
| Live | 30-60 min conversation, on-camera chat | Trust + real-time feedback |
| Post-live (24h) | Thank-you post, tag guests, clip highlights | Golden-hour + replay distribution |
| Repurpose (week) | 3-5 clips, 1 podcast episode, 2-4 feed posts | Feed algorithm + owned audio |

## When to host vs guest (pair with van der Blom)

Richard van der Blom's Algorithm Insights Report notes LinkedIn Live is
**infrastructure-heavy** — default advice is **guest frequently, host rarely**
for reach efficiency.

Lizak's counter-position (for founders/CMOs with a content mandate):

| Profile | Recommendation |
|---|---|
| Rep building pipeline | Guest on others' Lives; clip appearances to feed |
| Founder / exec with weak content ops | Weekly hosted show as **primary engine** — one live replaces a week of writing |
| Company with multiple SMEs | Rotate internal hosts; centralize production with one show runner |
| Team already posting 3-5x/week | Add Live as **anchor**; repurpose into carousels (top feed format per van der Blom) |

Hosting works when you commit to **cadence + repurposing**, not one-off broadcasts.

## Show formats that work for B2B

| Format | Best for | Production bar |
|---|---|---|
| **Weekly wins / lessons** (WinsDay model) | Operators sharing tactical wins | Low — solo or co-host |
| **Executive interview** | Complex product/story education | Low — two chairs, one mic |
| **Customer / practitioner spotlight** | Social proof + ICP empathy | Medium — brief prep doc for guest |
| **Panel / roundtable** | Category authority | Medium-high — moderator required |
| **AMA / office hours** | Community trust, CS, PLG | Low — FAQ backlog as outline |

**Rule:** If the conversation would be valuable as a podcast episode, it is
valuable as a LinkedIn Live.

## Pre-live promotion checklist

- [ ] Event created on LinkedIn with clear title + ICP keyword in description
- [ ] Teaser post 3-5 days before (no external link in body — `linkedin-algorithm`)
- [ ] Reminder post day-of with time zone
- [ ] Personal DMs to 10-20 warm contacts (not mass spam)
- [ ] Guest shares to their network if applicable
- [ ] Talking points doc shared with guest (not a rigid script)

## Live execution checklist

- [ ] Start on time; acknowledge commenters by name in first 5 minutes
- [ ] Pin a welcome comment with topic + how to ask questions
- [ ] Keep segments 8-12 minutes; verbal chapter marks for clipping
- [ ] Ask one audience question per segment (poll or comments)
- [ ] End with single CTA: follow, subscribe, book, or next episode date
- [ ] Save replay; download for podcast feed if applicable

## Repurposing map (minimum viable)

From one 45-minute Live:

| Asset | Destination | Notes |
|---|---|---|
| 3-5 vertical clips (60-90s) | LinkedIn native video, optionally Shorts/Reels | Hook = guest quote or tension statement |
| Full audio | Podcast RSS (WinsDay model) | Transcript → SEO + newsletter |
| 1 carousel | LinkedIn document post | Framework from conversation (~1.39x reach) |
| 2-4 text posts | LinkedIn feed | Stories, quotes, lessons — golden-hour replies |
| 1 newsletter | LinkedIn Newsletter or email | Deep dive on top thread |

**AI role:** transcription, draft posts, clip captions — not replacing the live
voice (van der Blom penalizes AI-sounding feed copy).

## Relationship discipline (non-negotiable)

Lizak's public emphasis: **real relationships beat algorithm tricks.**

- Reply to every comment on live teaser, live thread, and replay post
- DM thoughtful follow-ups to active commenters (not pitches)
- Re-invite engaged viewers as future guests
- Track "conversation quality" not just viewer count

## Measurement (practical)

| Metric | Why it matters |
|---|---|
| Live attendees + replay views | Reach of the engine |
| Comment count + unique commenters | Relationship depth |
| New connection requests post-show | Network growth |
| DMs referencing the episode | Pipeline signal (dark social) |

## Common failure modes

1. **One-and-done** — no weekly cadence; audience forgets the show exists.
2. **Over-produced** — delays launch; Lizak's insight: hit record on internal genius.
3. **No repurposing** — Live without clips/posts wastes the interest-graph window.
4. **Pitch theater** — product demo disguised as conversation; engagement collapses.
5. **Ignoring chat** — Live is bidirectional; monologue broadcasts underperform.
6. **Guest with no promotion** — split reach responsibility with co-marketing.
