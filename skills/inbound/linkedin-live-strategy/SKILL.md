# LinkedIn Live Strategy

## Overview

B2B teams burn budget manufacturing posts nobody asked for while their best
insights live in Slack threads, customer calls, and executive hallway
conversations. **Jessie Lizak** (Founder, **Reveting**; host of **WinsDay**)
flips the model: run a **weekly LinkedIn Live** as the primary content engine,
capture real conversations, and repurpose one session into podcasts, clips,
carousels, and feed posts for the whole month.

This skill sits beside `linkedin-algorithm` (Richard van der Blom's feed
mechanics). Van der Blom's data says Live is infrastructure-heavy — **guest
often, host rarely** for reach efficiency. Lizak's playbook is for founders,
CMOs, and execs who **will** host because a consistent show replaces a content
team's output — when you pair Live with disciplined repurposing and relationship
follow-up.

## Prerequisites

- Personal LinkedIn profile with Creator mode / Live access enabled
- Defined ICP and 2-3 content pillars (see `founder-brand`, `social-media-strategy`)
- Basic stack: LinkedIn Live-capable tool, recording, transcription (optional AI for drafts)
- Commitment to **weekly cadence** for 8 weeks minimum before judging ROI

## Step-by-Step Process

### Phase 1: Decide If Hosting Is Worth It

1. **Guest-first test (2-4 weeks):** Appear on 2-3 others' Lives; clip best moments
   to feed — lowest infrastructure path (van der Blom default).
2. **Host if:** founder/exec is the brand, content ops are thin, or you can assign
   one show runner (Reveting model).
3. **Kill if:** no one will own weekly prep + repurposing — one-off Lives waste effort.

### Phase 2: Design the Show (WinsDay Pattern)

1. Pick a **recurring slot** (e.g., Wednesday noon) — consistency beats perfection.
2. Name the show with an ICP keyword (marketing, RevOps, data, etc.).
3. Choose format: solo wins, executive interview, customer spotlight, or panel.
4. Build a 12-week guest/topic backlog before episode 1.
5. Document 3-5 talking points per episode — conversation guide, not script.

### Phase 3: Pre-Live Promotion

1. Create LinkedIn Event with keyword-rich description.
2. Teaser post 3-5 days prior (no external links in body).
3. Day-of reminder with time zones.
4. DM 10-20 warm contacts personally — not blast invites.
5. Co-promote with guest's network when applicable.

### Phase 4: Live Execution

1. Open by acknowledging live commenters; pin welcome comment.
2. Segment into 8-12 minute chapters for easy clipping.
3. Pull one audience question per segment from chat.
4. Close with one CTA: next episode date, newsletter, or booking link in **comments**.
5. Within 24 hours: thank-you post tagging guest + top commenters.

### Phase 5: Repurpose (Non-Optional)

Minimum from each session (see full table in reference):

- **3-5 native video clips** (60-90s hooks) for LinkedIn feed
- **1 document carousel** summarizing the framework discussed
- **2-4 text posts** (quotes, stories, lessons) with golden-hour replies
- **Podcast/audio feed** if building owned audience off-platform
- Use AI for transcription and **drafts** — human-edit voice before posting

### Phase 6: Relationship & Pipeline Follow-Up

1. Reply to every live and replay comment in the first 60-90 minutes.
2. DM engaged commenters with value, not pitches.
3. Invite active participants as future guests.
4. Log self-reported "heard us on Live" in CRM for 30-day review.

## Common Pitfalls

1. **Treating Live as a one-off webinar.** Without weekly cadence, the algorithm
   and your network forget the show exists.

2. **Over-production paralysis.** Lizak's origin story started with CEO
   interviews and a record button — not a studio budget.

3. **Skipping repurposing.** One Live without clips and carousels leaves feed
   reach on the table (carousels ~1.39x per van der Blom).

4. **Monologue mode.** Ignoring live chat wastes the format's trust advantage.

5. **Pitch theater.** Product demos dressed as conversations kill comment depth.

6. **AI voice on repurposed posts.** Draft with AI, edit for human voice — AI-sounding
   feed copy is penalized in van der Blom's data.

7. **Measuring only viewer count.** Comment quality, DMs, and 30-day meetings
   matter more than peak live attendance.
