## Core thesis: filter-specific messaging, not one script

Default Sales Navigator failure mode: build a list, blast the same message to
everyone. Ingram's fix: **each filter signal gets its own message** — the filter
is the personalization hook.

| Old way | Ingram way |
|---|---|
| One template × 4,000 sends | 12+ filter-tied message variants |
| ~1% response (generic benchmark) | ~5% response when filter + message match |
| Sales Nav as list export | Sales Nav as **intent engine** with weekly alerts |

Directional team math (public post): 10 reps × 20 messages/day × $20K ACV —
generic ~$200K pipeline/month vs filter-specific ~$1M at stated response
assumptions. Treat as illustration, not a guarantee.

## High-intent Sales Navigator filters

Stack these on title, seniority, geography, and company size for ICP fit.

| Filter | Signal | Message angle |
|---|---|---|
| **Posted on LinkedIn in last 30 days** | Active on platform; 3× exec response lift in Ingram's testing | Reference their post; insight + question before connect |
| **Following your company** | Brand-aware; warm | "Noticed you're following [company] — curious what prompted the follow?" |
| **Viewed your profile** | Already checked you out | "Saw you stopped by my profile — happy to share [resource] if useful" |
| **Years in current company: < 1 year** | New-role transition pain | "Talking to [title]s 6–7 months in — [challenge]. Seeing that too?" |
| **Years in current company: 6–12 months** | Past onboarding; buying window | "Month 6–9 is when teams usually [initiative]. On your radar?" |
| **Changed jobs in last 90 days** | Champion / new broom | Congratulate + role-specific pain hypothesis |
| **Connections of** (target exec) | Warm path via mutual | Ask mutual for intro or reference shared connection |
| **Account follows your company** | ABM warm account | Multi-thread with filter-specific plays per persona |

**10-minute setup:** Save search with VP/C-suite + Posted 30 days + Following
company + Viewed profile + tenure filters → enable weekly alert (green button).

## Executive engagement sequence (posted-in-30-days)

1. Filter Sales Navigator: Posted last 30 days + ICP titles + seniority + geo.
2. Open prospect's **most recent post** (not profile pitch).
3. Comment with **insight + one question** — not "great post!"
   - Example pattern: "Interesting take on [topic]; we've seen [adjacent pattern]
     in [industry]. What prompted your team to focus on [specific point]?"
4. **Wait 24–48 hours** — let notification land; avoid instant connect spam.
5. Send **connection request** referencing the thread (no pitch).
6. After accept → **DM** continues the conversation; soft CTA only after value.

**Time budget:** ~15 minutes/day for this motion (Ingram public claim).

## Filter → message library (build 12 variants)

| Filter bucket | Opening hook template |
|---|---|
| Posted recently | "Your post on [topic] — especially [specific line] — …" |
| < 1 year tenure | "Was talking to a [title] who's [N] months in. They said [pain]. Are you seeing that?" |
| 6–12 months tenure | "Month [N] is when most [title]s start [initiative]. Where are you on that?" |
| Following company | "Noticed you've been following [company]. What caught your attention?" |
| Viewed profile | "Thanks for stopping by my profile. Were you researching [topic]?" |
| Job change 90d | "Congrats on the new role at [company]. Curious how you're approaching [priority]?" |

Store variants in CRM or sequencing tool; never recycle one script across filters.

## Boolean / decision-maker search

- Combine title Boolean (`VP OR Director OR Head`) with function keywords.
- Exclude irrelevant functions (HR, IT) when selling to GTM buyers.
- Use **Lead filters** + **Account filters** separately, then intersect lists
  for ABM accounts.

## Multi-threading on accounts

For target accounts, run **parallel filter plays** per persona:

| Persona | Filter emphasis | Goal |
|---|---|---|
| Economic buyer | Posted 30d + C-suite | Executive DM thread |
| Champion | 6–12 months tenure + Following company | Operational pain discovery |
| Technical | Posted on technical topics | Credibility comment |

## Common failure modes

1. **Generic "great post" comments** — no conversation starter.
2. **Connect request same day as comment** — feels automated.
3. **One message for all filters** — wastes Sales Nav signal data.
4. **No saved searches** — reps re-run manual searches weekly (waste).
5. **Pitch in connection request** — lowers accept rate.
6. **Ignoring profile views / company followers** — highest-intent free signals.
