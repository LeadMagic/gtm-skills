# LinkedIn Algorithm Audit — Deliverable

## Deliverable spec

LinkedIn algorithm audit + optimization plan: 90-day baseline table, reach-killer
findings, format mix recommendation with multipliers, post anatomy checklist
(hook/length/keywords/tags), golden-hour engagement routine, posting calendar
(frequency + timing), and a 30-day remeasurement plan.

## Quality check

- [ ] Baseline separates platform-wide compression from personal decline
- [ ] Every recommendation cites a report finding, not folklore
- [ ] Format mix includes document posts for save-worthy content
- [ ] No external links in post bodies; links moved to comments/profile
- [ ] Hashtags removed or capped; keywords written into body copy
- [ ] Golden-hour engagement routine scheduled, not aspirational
- [ ] 30-day remeasurement date set with the same metrics
