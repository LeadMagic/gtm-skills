# Onboarding Sequences — Framework Notes

Reference tables for `SKILL.md`. One job: get customers to the aha moment in week 1.

---

## Primary frameworks

- **Lincoln Murphy Desired Outcome** — Define success before designing steps
- **Gainsight Onboarding Framework** — Milestone-based CS onboarding
- **Reforge Lifecycle Marketing** — Activation correlates with 90-day retention

---

## Aha moment examples by product type

| Product type | Aha moment | Retention correlation |
|---|---|---|
| Data/enrichment | First successful enrichment | 80%+ 90d if hit in 7d |
| CRM | First deal stage change | |
| Analytics | First dashboard built | |
| Sequencing | First sequence sent | |

**Target:** 80%+ of new customers hit aha moment within 7 days.

---

## Onboarding journey map

| Day | Milestone | Channel | Content |
|---:|---|---|---|
| 0 | Pre-launch prep | Email | Welcome + prerequisites |
| 1 | First action | In-app + email | Guided single action |
| 3 | First value check | Email | "Did you try X?" |
| 7 | Second value | Email | Advanced use case |
| 14 | Habit check | Email | Usage stats vs peers |
| 30 | Expansion | Email | Power features |

---

## Multi-channel mix

| Channel | When | Rule |
|---|---|---|
| In-app | Always | Show 3 steps to aha — not full product tour |
| Email | Behavior-triggered | Step N complete → email step N+1 |
| CSM call | Enterprise | Week 1 kickoff screen share |
