## Authoritative foundations

Enablement succeeds when **one message** is operationalized: discovery questions,
demo scenes, objection responses, and CRM fields all map to the same Value
Messaging Framework. Avoid content libraries nobody uses — certify reps on
behaviors (Barrows drills), not slide decks alone.

## Process phases

- **Phase 1 — Message architecture:** VMF / positioning → required capabilities per persona
- **Phase 2 — Asset build:** Talk tracks, demo scripts, battlecards, email templates
- **Phase 3 — Certification:** Role-play + call review (Dorsey/Barrows cadence)
- **Phase 4 — Reinforcement:** Monthly refresh; win/loss feeds back to assets
