# Contributing

The canonical contribution guide is at [`../CONTRIBUTING.md`](../CONTRIBUTING.md).

Use this short checklist before opening a PR:

```bash
npm run build
npm run check
gh skill publish --dry-run
```

For skill authoring rules, see [`SKILL_AUTHORING.md`](SKILL_AUTHORING.md).
For repository architecture, see [`ARCHITECTURE.md`](ARCHITECTURE.md).
For release steps, see [`RELEASE_PROCESS.md`](RELEASE_PROCESS.md).
